第一部
017　美琴☆☆
淫靡な音が部屋に響いていた。

床にしゃがんだ僕。

その股間に美琴が顔をうずめ肉棒を舐める。

右手で一所懸命にしごきながら肉棒を舐める。



「んっ……ちゅぷ、ん」



あまりの可愛さに僕は美琴の頭を撫でていた。

美琴はただ集中して目の前にある太い棒を舐めつづける。



「ん、ちゅぷ、くちゅ、ちゅぷ、んっ……はむ」



亀頭を口に咥えた。

そして舌を使ってぐりぐりと舐めまくる。

僕の頭の中はぐちゃぐちゃだった。

なにか考え事をしようとすると白い快感の波によってそれら考えはさらわれていてしまう。



「美琴……美琴……」



いつの間にか無意識に僕は美琴の名前をつぶやいていた。

それに呼応するかのように美琴の動きが激しくなっていく。



「ん、ちゅぷぷ、くちゅんっ、んぱっ、ん、あちゅぅ」



しごく手の速度があがっていく。



「ああああ」



射精が近いことを腰のふるえが伝えた。



「美琴！　イっちゃう！」



僕は美琴の頭を軽くおした。

亀頭から口を離した美琴が僕の目を見る。

真顔だ。だけどそこにはどこか哀愁の影があった。



美琴はしごく手をとめなかった。

じっと僕の目を見たまま肉棒をしごきつづける。

濡れた唇を噛んで、なにかを訴えるように手の速度をあげていく。



「イクっ！　イクっ！」



いままでに感じたことのない快感の塊が細い管を通っていく。

腰が床から浮き、そそり立った肉棒が美琴の顔へと接近した。

僕のことを見ている美琴。その顔に勢いよく白濁液が飛んでいった。

一気に開放された快感の塊が射精と同時に弾け飛ぶ。



「んんんっ」



ほとんどすべての精液を美琴がそのちいさな顔で受け止めた。

しかし僕から目を離さなかった。



「はっ……はっ……はっ……」



あまりの快感と解放感に、僕の肺は空気を失っていた。

心拍数が異常にあがった心臓の鼓動が全身で響く。

体が二度軽く痙攣した。



「み、美琴……」



目に鼻にそして口にべっとりとついた精液が重力にも負けずに美琴の顔を汚している。

ふっと美琴が口元を緩めた。驚くほどに純粋な微笑みだ。



「……ねえ、また、して欲しい？」

「え……あ、ああ」



こんな快感があるなら僕は美琴の奴隷になってしまうかもしれない。



「なら約束して……だれとも付き合わないって」

「え？　ど、どういうことだ……？」

「わたしも、だれとも付き合わないから……」



どういう意図でそんなことを言うのかは定かではない。

美琴は立ちあがるとティッシュを手にして顔を拭いた。

そしてそのティッシュをゴミ箱に入れると狭い部屋を横断した。

扉まで行き、鍵をあける。ドアノブを回した。

こちらに背をむけたまま言った。



「約束だからね……」



逃げるように美琴は部屋を出ていった。

廊下を去っていく足音と快感の余韻だけを残していなくなった。



次の日。

僕は朝に美琴と会った。

グレーのワンピースの制服を着てお嬢様学校へと通う美琴は、昨日のジャージ姿の美琴とは違う人間のようだった。



「おはよう」

「おはよう」



それだけを言い交し、駅まで二人とも黙って自転車をこいだ。

乗る電車が反対のため、改札を抜けたところでわかれる。



「じゃ」

「うん」



いつもの朝だった。

美琴と一緒に駅まで来ることはたまにある。

時間が合えば一緒に駅まで行くのだ。

そのとき会話が弾んだためしはない。

ひとつは二人ともに朝が弱いことがある。

ほとんど毎日会っているため朝にわざわざ会話をする必要もないのだ。



昨日は異様なことがあった。

しかしそれでも日常はつづいていくようだった。



授業中にやはり来栖と何度も目が合った。

来栖がみんなに囲まれて談笑しているとき、先生に指名されて立つ瞬間、体育のバスケットで来栖が見事なシュートを決めたてみんなと喜びを分かち合っているとき、事あるごとに僕と来栖は目線を送り合っていた。

放課後になりトイレから教室にもどる途中で来栖とすれ違った。



「ちょっと」

「ん？」



来栖は一人だった。

さすがにトイレに行くときはとりまきを置いてくるらしい。

足をとめた僕と一定の距離をおいて来栖が独り言のようにつぶやいた。



「見すぎだよ」

「え？」

「ほら、授業中とか体育のときとかさ……」



すごい恥ずかしいことを指摘された。

僕は自分の顔が赤くなるのがわかった。



「ごめん……」

「イヤとかそういうんじゃないんだよ？」



来栖は僕と目を合わせない。



「でも困るんだよね」

「そうだよな……気を付ける」

「うん……わたしもさ、奥谷くんが見てるんじゃないかって、いちいち気にしちゃうから……」



照れたように来栖も顔を赤らめた。



「それで、わたしのこと見てなかったとき、ガッカリとかしちゃうからさ……」

「あ、ああ」



そういうことか。

来栖が可愛すぎていまにも襲いかかってしまいそうだった。



「だから、ちょっと目合わすのさ……」

「やめるか？」



首をふると来栖が僕の目を見た。



「やめない！　でも、すくなくはしよ？」



完全にやめる気はないらしい。

このままではますます目を合わす頻度は多くなりそうだった。

そこのあなた、ブックマークをお忘れじゃないですか？
