10/295
一話一話が短くてすみません。

かわりにテンポを重視してます。そのはず。

第一部
010　美亜
次の日。つまり日曜日。

クリーニング屋へと制服を受けとりにいった。

すでに美琴の制服は本人によって回収されていた。

あれから電話とメール、直接に家まで行ったのだがすべて無視された。



「なにやってんの？　高校生にもなって喧嘩？」



直接、家へと訪ねたとき呆れながら美琴母が対応してくれた。

勢いあまったとはいえあれは僕が悪かったのだ。

ちゃんと謝罪をしたかったのだが、いまはそっとしておこう。



で、さらに次の日。つまり月曜日。

これから一週間、また学校へ行くことになる。



僕のチンコはたしかにデカい。デカチンだ。

だが、美琴が言うには変というわけでもないらしい。

気持ち悪いわけでもないとのことで一安心。

処女が言うことだからあてにはならないがそれでも希望はあった。



しかし問題はなにも解決していない。



来栖美亜のことだ。

金曜日の放課後、僕は来栖に自分の秘密を暴露した。

引くほどチンコがデカいと言ってしまった。

美琴に変じゃないと言われたところで、来栖へ暴露したという事実は消えない。

やはり冗談だったと告げ、許してもらうしかないだろう。



教室に入るとすでに来栖はいた。

来栖のまわりには女子も男子も大勢の生徒が集まっている。

他のクラスからも来ているらしく、僕のことに気付く様子はない。

ほんと人気者だ。



授業中に目が合うことはなかった。

まあ、それはいつものことだ。

気にすることでもない。

はなしをしないのもいつものこと。



放課後。

部室に行くと来栖はいなかった。

先に教室を出たのを見たから、てっきりもういると思っていた。

部長の川内に尋ねてみると、ひどく悲しそうな顔で首をふった。



「やめたのか？」

「違う……体調不良で休むってさ……」



絶望するほどのことでもないだろう。

しかし来栖がいない文化人類研究部はひどく沈んでいた。



来栖は体調不良には見えなかった。

いつも通りに明るく元気で健康的であった。

部活を休んだ理由は僕にあるに違いない。



「僕もちょっと……」



理由も言わず僕も部活を休むことにした。

他の部員はまったく気にする素振りもみせなかった。



早いほうがいい。

このまま日数が経てばどんどん来栖の中で僕のチンコは肥大化してしまう。

そうなればせっかく友達になれたのに関係は破たんするだろう。

杞憂かもしれないが、なにもしないよりはいい。



バスに乗り駅へ。

来たばかりの電車へと飛び乗る。

三倉駅でおり、来栖が住むタワーマンションへとむかった。



勢いだけで来てしまった。

だが引きかえすわけにもいかない。

オートロックの前でうろ覚えだった部屋番号を押す。

しばらくして声がした。来栖母のものだった。



「はい……」

「ああ、すみません。先日お邪魔した奥谷です。美亜さんはご在宅ですか」

「あらこんにちは。ちょっと、待ってね」



一分もしないで来栖母の声がかえってきた。



「そこで待っててほしいって、下におりるみたい」

「わかりました」



待つあいだ時間が長く感じた。

このまま走って逃げてしまいたい衝動にかられる。

でも、それでは駄目だ。

ちゃんと冗談だったと伝えなくてはいけない。

実際には時間にして五分も経っていなかっただろう。



「ごめん、お待たせ」



軽く手をあげて来栖がやってきた。

制服のままだ。

学校ではおろしていた髪をいまは二つに結んでいる。

それがすこし来栖を幼くみせてドキドキした。



「お、おう……大丈夫か？　体調不良だって？」

「まあ……」



嘘をついた罪悪感からだろう。

来栖は曖昧な態度だ。



「心配になって、来てくれたの？」



来栖が尋ねる。

僕は首をふった。



「いや……一昨日のことで……」

「うん」



わかっていたのだろう。

うなずくと、来栖が歩き出した。



「すこし歩いたところに神社があるじゃない？　そこ行こう」



その神社は僕も知っていた。

駅から歩いて五分ほどの場所にある米島神社。

雑木林にかこわれた、ひどく廃れた神社だ。

先を歩いていく来栖に僕は声をかけた。



「来栖……あれは冗談だったんだよ」



しかし返事はない。

怒っているのか、それとも幻滅したのか。



「なあ……なんか言ってくれよ」



結局、来栖が口をひらいたのは神社についてからだった。

境内の裏と雑木林とのあいだ、砂利が敷かれた薄暗い場所。

夜になるとカップルが逢引に使うような人目から隔絶された場所であった。

遠くで下校している小学生の甲高い声がきこえてくる。



「見せて」

「はい？」

「だから……奥谷くんの、その……」



この展開は予想外すぎる。

美琴は言っていた。

女子なら見せてと言わないから安心しろ、と。

だが、来栖はたしかに言った。



「奥谷くんのチンチン……見せて」

「あ、あ……」



恐れていた事態となった。

数日のあいだに二人の女子からチンコを見せるよう要求されている。

異様なことだ。



「わたし、あれからずっと考えてたの……」



来栖がつぶやくように言う。

顔が真っ赤だ。

目が潤んで瞳は焦点が定まっていない。



「奥谷くんは秘密を教えてくれたのに、わたしひどい態度をとったなって」

「あ、そ、それは……気にしなくていい。あれは冗談だったんだ」



ちゃんと伝えなくては駄目だ。

だが、来栖が首をふった。



「違う！　奥谷くんは咄嗟にあんな嘘を吐けるような人じゃないでしょ？」



僕のなにを知っているのだろう。

しかし来栖の目は真剣だ。



「それに嘘はつかないで欲しい……私は奥谷くんとは本当の友達になりたい。なったと思ってる。だから、嘘はつかないで欲しい」

「……わかった」



そこまで言われてしまえば抗えない。

このあとどうなろうと僕は後悔しないだろう。



「あれは、冗談じゃない……」



訂正した。

来栖に嘘は言わない。

いまそう決めたのだ。



「たしかに僕のチンコは引くほどデカい……」

「そう……」



やっと笑顔を見せた来栖。

しかしその笑顔は照れ隠しにも見えた。



「お、奥谷くん……見せてくれない？」

「どうして見たいんだ？」



その質問にしめっぽい声で来栖が答えた。



「奥谷くんの言葉を疑っているわけじゃないよ？　だけど秘密はちゃんと共有しないと……」



つまりちゃんとチンコがデカいか確認する。

そしてはじめて僕の秘密を来栖は知ったことになるということだ。

滅茶苦茶な論理だが、たしかに僕のチンコがデカい証拠はない。

実際に見せるのが一番てっとり早いだろう。

僕は諦めた。



「わ、わかった……」



来栖とは友達になったのだ。

見せて、どんな結果になろうともそれが運命だと受け入れよう。



雑木林がさわさわと風に揺れている。

日が沈みはじめるまでもうすこしあるだろう。

しかし人の気配は皆無だった。

きこえていた小学生の声もいつの間にか途絶えていた。



「じゃ、じゃあ……」



意を決して僕はズボンを脱いだ。
