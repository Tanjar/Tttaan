016　美琴☆☆
「美琴、おい、やめろって！　あうっ」



僕は携帯電話のマイク部分を手で覆い美琴へと叫ぶ。

両手でしっかり僕の足を掴んだ美琴が使えるのは口だけだ。

しかしだからといって、いきなり口にふくむとは恐ろしい。



「ん……でっか……ん……」



通常時であっても僕のチンコはデカい。

それは日本人が勃起したときと変わらない長さと太さだ。

美琴は僕のチンコを口に入れたまましゃべる。



「これ、ん……大きくなったら……無理だよ……」

「やめろって！　なんで、こんなことすんだよ！」



しかし美琴は答えない。

口内の温かさが肉棒から這いあがってくる。

腰が引けたが、逃げようとすればするほどに美琴が深く口へとふくむ。



「すまん……うちの猫が」



しかたない。

なにかいい訳をしてさっさと城田との電話をおえよう。



『え？　奥谷の家って猫飼ってるの？』



飼ってない。



『わたしさ、猫、好きなんだよね、うちにも二匹いて』



まずい。

はなしが長くなりそうだ。



「ふぁんっ」

『え？』



変な声が出てしまった。

美琴が肉棒の先を固くした舌で舐めたのだ。

冷や汗が出るほどに気持ちがいい。

さっさと快感に溺れてしまえと僕の中にいる悪魔が囁く。

天使のほうは不在だ。自力で踏ん張るしかない。



『どうしたの？　猫に舐められた？』



城田が都合のいい解釈をする。

まあ、間違ってはいない。

美琴は猫のようなやつだ。



「そ、そうだ……んっ、馬鹿、やめろ」

『かわいい猫？』



気持ちがいい。

とうとう肉棒が勃起を開始した。

それに気づいた美琴が目を見開く。



「か、かわいいよ……」



美琴はかわいい。

思わず携帯電話を持っていないほうの手で美琴の頭を撫でていた。

大きくなりつつある肉棒をくわえながら美琴が照れくさそうに目を伏せた。



『ねえ、うちの猫も見たい？』



城田が質問してくる。

簡単な返事しかできない。



「あ、そ、そうだな……」



くちゅん、くちゅん。

美琴が唾液をふんだんに使って亀頭部分を舐めまわす。

ふわりと体が浮いたような感覚に襲われ、肉棒がただ快感を貪る器官となる。



『ならさ写真送ってあげる！　いまね、わたしの部屋にいるから。ちょっと、待ってて』



うれしそうな声をあげて城田が電話を切った。

こちらが努力しなくても勝手に切ってくれた。

とりあえず安心だ。



問題は肉棒をくわえる美琴だ。

かなり勃起は進んでいて美琴もくわえるのが限界なのだろう。

すこし苦しそうに表情を歪めている。



「もういいから、口、離せよ」



しかし僕の優しさは無視された。

美琴が首をふって、うるんだ目で僕を見る。

そしてなにか言おうとした。



「んんんん……んんんん、んんんん」

「その状態でしゃべるな！　可愛いな、もう！」



携帯電話を机に放ると、僕は美琴の頭を両手ぐしゃぐしゃにした。

イヤそうに首をふりながらも美琴は肉棒をはなさない。

一所懸命に小さな舌を亀頭に押しつけながらどうにか刺激を与えようとしてくる。



「美琴……気持ちいぃ……」

「……んんんん……はっ」



さすがに限界だったらしい。

肉棒を口から離すと、美琴が僕を睨んだ。



「ちょっと！　顎、外れるから！　なにこれデカすぎ！」

「ごめん」

「でも、大きくなった……」



僕の足から手を離し、美琴が巨大化したチンコを指先で触れた。

ぴりぴりと微弱な電気を受けたような刺激が走る。



「う、動いた……すごい……」



興味深げに肉棒を眺める美琴の口元は唾液で照っていた。



「美琴……」

「なによ？」



机の下にいる美琴と目が合う。

本当に猫のようだ。名前を呼ばれて目を丸くした猫だ。



「出てこいよ、そこ狭いだろう？」



僕は立ちあがると椅子をずらした。

四つん這いでジャージ姿の美琴が机の下から出てくる。

そして膝立ちになった。もちろん目の前には僕の息子が待っている。



「それで？」



すこし目を細くして、僕を睨む美琴。

頭をくしゃくしゃにされたために美琴の短い黒髪は乱れきっている。

顔は首のほうまで真っ赤だ。机の下にいたためにうっすらと汗をかいていた。



「握ってくれ……」

「はあ？」

「握って、先っぽを舐めてくれ」

「なに言っての、馬鹿じゃん！」



たしかに僕はなにを言っているのだろうか。

そう思ったとき携帯電話がメールの着信を告げた。

一気に冷静になる。



「そうだな、馬鹿だな、僕……」



手を伸ばして、机の上にある携帯電話を手にする。

城田からのメールだろう。



「ごめん……忘れてくれ」



美琴は不機嫌そうに僕をじっと見ていた。

さすがに僕が言ったことに怒っているらしい。

そんな美琴の目から逃れるために、僕はメールを見た。



＞じゃーん(#^.^#)

＞我が家のかわい子ちゃんたちでーす♪



その文言のあとに写真が添付されていた。

丸々太った黒猫と細身の茶色い猫がベッドの中で絡み合うように寝ている写真だ。

城田の部屋で撮られたものだろう。

猫を撫でる日に焼けた城田の腕も映りこんでいた。



「ああああ！　もう！」



声がした。美琴の声だ。

はっとして見てみる。

美琴が頬を膨らませて僕を睨んでいた。

かなり怒っている。

しかしもとはと言えば美琴が無理やりに僕のチンコを咥えたのが悪いのだ。



「こうすればいいの？」



怒った顔のまま美琴が肉棒の幹を握ってきた。



「おい！」

「いいから黙って！　もう黙って！」



叫んでから美琴が肉棒を握ったまま先っぽを舐めはじめた。

チロチロと尿道部分を重点的に舐める。

寒気にも似た快感が腰のあたりを覆った。

顔は怒っているのに舌の動きは健気で一所懸命だ。

その矛盾した行動に僕は妙に興奮をおぼえていた。



「んちゅ……ん、んあ、ん」

「ああ、美琴、しごいてくれ」

「え？　なに？　しごく？」



口調はつっけんどんだが、僕と見つめ合った美琴の目には喜びが滲んでいた。



「こう……手を動かして欲しい」



僕は持っていた携帯電話を肉棒にみたせて説明した。

眉を潜めながらも美琴が言われたとおりに肉棒をしごきはじめる。

自分の唾液で濡れに肉棒を美琴自身がしごいている。

すごく扇情的な光景だ。



「舐めてくれ」



気持ちがよかった。

美琴の手はちいさい。

僕の勃起したチンコを完全に握ることはできない。

しかしその刺激のムラが興奮を助長させた。



「ちゅ、ん、ん、ん」



すこしだけしか口の外に出ない舌で、美琴が亀頭を舐める。

腰が引けてしまうほどに刺激が強かった。

最近まで異性として意識したこともなかった幼馴染。

その美琴がいまは僕のチンコをしごいて舐めている。

あまりの事態に脳がパニックをおこしそうだった。



「あ……気持ちいぃ……すごく、いい」

「ん、ん、ちゅ、ん、ちゅ」



舌だけを動かすのに疲れたのだろうか。

美琴が顔も一緒に動かす。



そのときだった。

手で握っていた携帯電話が鳴った。

また返事がないから城田から電話だろうか。

そう思ったが違った。メールだ。



＞そうだ！　明日うちに来る？

＞そうすれば猫が見られるよ？

＞そうだ、そうしよう決定('ω')ノ



思わずメールを見てしまった。

するとそれを機に美琴が手の速度を速めた。

そして亀頭部分をばくりと口にふくんだ。



「おおおおっ？」



急な刺激に腰が砕けて、僕は床に腰をおろした。

手から携帯電話が落ちる。



「美琴……」



美琴が四つん這いになり、僕の肉棒へと近づいてくる。

なにかに取りつかれように肉棒の幹を掴むとまたしごきはじめた。

僕はただ美琴の行動を享受するしかなかった。

つづきが気になる！　という方はぜひともブックマークをお願いいたします。
